# Urban Transit Platform
[中文](README.md) | [English](README_EN.md)


(目前)一个基于 Python（后端） 和 TypeScript + React（前端） 的火车票信息爬取工具，用于获取并展示 CR 的列车与车票相关信息。
致力于实现精确到分钟的大湾区公共交通通勤方案
## 📌 项目概述

Train Crawler Tool(曾用名) 是一个全栈应用，主要功能包括：

- 从 CR 相关数据源中爬取列车与车票信息

- 在后端对车票数据进行处理、解析与结构化

- 提供基于 Web 的前端界面，用于数据展示与交互

## 📝 未来展望
- 基于 **Dijkstra** + **换乘惩罚** 实现湾区城市内的换乘最优路径

- 基于**实时爬虫** 访问铁路/城际系统的票务信息，寻找跨城市间的通勤策略

- 综合二者的总体时间惩罚与用户提供的策略(比如指定火车车站/换乘少优先/金额低优先/...)实现大湾区交通实现方案的提供

- 基于**AI Agent**输出自然语言换乘方案，通过LLM实现对于用户策略的分析与工具的调用
